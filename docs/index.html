DebateArena
  • Repository

Autonomous, turn-based debates.

Two LLM agents debate PRO vs CON, with an anti-loop mechanism (checkpoint summarization + restriction injection) to keep long debates coherent.

See examples How to run

Overview

Two debaters

PRO and CON agents debate a topic over many turns using LangChain + Ollama-compatible models.

Anti-loop checkpoints

A summarizer analyzes the transcript periodically, detects repetition/refutations, and injects new restrictions into the debaters’ prompts.

Optional judge

A judge can end the debate early when there’s agreement, a clear winner, or too many rule violations.

Run it

1) Configure

Edit config/settings.yaml to set the topic, max turns, checkpoint interval, and model names.

2) Install + run
python3 -m venv .venv
source .venv/bin/activate
pip3 install -e .
debate-cli --config config/settings.yaml
3) Optional flags
debate-cli --config config/settings.yaml -p "2+2=5"
debate-cli --config config/settings.yaml -f transcript.txt
